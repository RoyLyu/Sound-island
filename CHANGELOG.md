# 更新记录

## 0.7.0 — UCS 分类与专业选择工作流

- 移除旧的九类启发式分类，接入 Universal Category System 8.2.1：82 个母类、753 个子类规则、中英文同义词及 CatID 均在本地匹配。
- UCS 侧栏改为母类/子类层级与独立滚动；匹配器先判断母类，子类证据不足时明确标记为「未细分」，避免用共享关键词强猜子类。
- 本地素材库排序改成清晰的上移/下移按钮，顺序继续持久保存到本地 SQLite。
- 音频列表新增右键菜单：复制真实文件对象、在声音实验室编辑、定位原文件与切换收藏，并支持 `Cmd/Ctrl + C`。
- 底部试听栏移除 Space、导出与音高按钮，延长音量条；新增 Mono、Stereo、Quad、5.1、7.1 和其他多声道徽章。
- 右侧声音详情新增突出的声道/格式卡片，并展示 UCS Category、SubCategory 与 CatID。
- 声音实验室波形提高到 420 个峰值柱并增加采样密度，独立进度条缩短以避免与波形视觉竞争。
- 新增对白聚焦、超重低频、梦境扩散和紧张脉冲四个高辨识度方案；与实时试听及 24-bit WAV 导出共享同一组参数。

## 0.6.0 — 流畅试听与静态波形

- 底部试听条改用动画帧直接更新走带和读秒，不再由播放进度触发整页 React 重绘。
- 主试听链移除实时频谱与 Web Audio 分析节点；播放期间只移动预计算波形的游标。
- 删除右侧声音详情中无内容的频谱框，让名称、中文显示名与音频规格直接进入视线。
- 声音实验室在音频载入时一次性生成轻量波形，播放时显示波形、游标并保留独立拖拽进度条。
- 声音实验室进度与时间改为动画帧直接更新，避免整套效果参数控件随读秒重复渲染。
- 进一步放大声音实验室的方案、参数、运输控制和导出文字，并提高次级文字对比度。

## 0.5.1 — 实时试听性能修复

- 修复 v0.5.0 重复发布：Release 只由新版本标签触发，避免同名安装包二次上传失败。
- 实时频谱从 8192 点降为 1024 点轻量 FFT，并限制在约 25 FPS；停止播放时不再持续刷新 Canvas。
- 声音实验室播放进度把整套 React 控件刷新频率限制在约 12 FPS，效果滑杆重建音频链增加防抖。
- 快速切换列表选择时延迟启动波形解码，避免堆积多个已过期的后台分析任务。
- 声音实验室新增可拖动试听进度条，拖动前后保持播放状态。
- 立体声拓宽、单声道立体化、空间迁移与隔墙遮挡改为与一键方案一致的左侧卡片入口。
- 提高声音实验室主要文字字号、字重和对比度，改善小字号模糊。

## 0.5.0 — 全库检索与素材治理

- 整体提高界面最小字号和次级文字对比度，保持暗色专业风格。
- 顶部搜索固定为全库检索；输入关键词时不再继承分类、子分类、素材库或文件夹筛选。
- 智能分类再次点击会同时收回子类并取消当前分类选择。
- 母文件夹展开后展示全部后代目录，并用层级缩进保留原始目录关系。
- 本地素材库支持拖动调整顺序，顺序写入本地数据库并在重启后保持。
- 底部播放器只保留完整波形；右侧声音详情改为高精度实时频谱。
- 声音实验室把立体声拓宽、单声道立体化、空间迁移和隔墙遮挡统一收进左侧方案区。
- 导入和重新扫描新增解码门禁：没有可解码音频帧的文件不入库，已索引的损坏文件会被摘除。
- 发布与 QA 工作流补充全库搜索、递归目录、排序持久化和损坏音频拦截检查。

## 0.4.0 — 空间工作流

- 修复中文显示名数据流：路径只定位记录，翻译输入明确限定为原始音频名称。
- 支持把列表音频作为系统文件拖出，外部接收为复制体。
- 新增 `F` 收藏、`Z` 撤回、`Tab` 聚焦搜索、`Cmd/Ctrl + E` 导出所选文件。
- 主播放器新增随 Space 试听实时变化的频谱，并保留完整波形与播放头。
- 声音实验室新增立体声拓宽、低频单声道保护、中置信号保持、相位相关度检测与兼容保护。
- 新增相位安全单声道立体化、11 种空间迁移和 7 种隔墙/遮挡方案。
- 实时试听与离线 24-bit WAV 导出共享参数；导出继续拒绝覆盖母文件。
- Release 工作流新增独立质量门禁，打包前执行前端构建、Rust 测试、格式及 Clippy 检查。

## 0.3.0 — 声音实验室

- 素材库与智能分类支持展开和收回。
- 强化 FTS5 搜索，新增 W/S 选择与基础声音实验室。
